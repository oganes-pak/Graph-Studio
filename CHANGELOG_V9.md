# Graph Studio v9

## Главное

Graph Studio v9 превращает прежний редактор сетевого графа в модульный редактор десяти типов диаграмм.

## Исправлен импорт

Импорт больше не останавливается на ошибке «связь уже существует».
Перед загрузкой выполняется `normalizeProjectImport()`:

1. повторные узлы объединяются по `id`;
2. `A → B` и `B → A` считаются одной парой;
3. повторные связи объединяются;
4. для числового `value` значения складываются;
5. петли и ссылки на отсутствующие узлы пропускаются;
6. пользователь получает отчёт об исправлениях.

Строгая ручная операция `add_connection` по-прежнему запрещает дубль. Мягкая очистка применяется только к импорту и полной замене проекта.

## Добавлены типы диаграмм

- `fishbone` — диаграмма Исикавы;
- `flowchart` — блок-схема;
- `tree` — дерево иерархии;
- `force` — силовой граф;
- `sankey` — потоки Санкея;
- `network` — исходная сетевая диаграмма;
- `radar` — радарная диаграмма;
- `bubble` — пузырьковая диаграмма;
- `mindmap` — ментальная карта;
- `decision` — граф решений.

## Машинные команды

Число атомарных инструментов увеличено с 13 до 15:

```text
set_diagram_type
set_chart_data
```

Одинаковые имена используются в браузере, Web Component и MCP-sidecar.

## Формат данных

Актуальный декларативный формат:

```text
graph-studio/2
```

Он сохраняет совместимость с импортом `graph-studio/1`.

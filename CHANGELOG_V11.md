# Graph Studio v11: демонстрации, совместимость и Bubble

## Главное

- Демонстрации всех десяти диаграмм заменены на насыщенные предметные сценарии.
- Добавлена видимая оценка смысловой совместимости данных с выбранной диаграммой.
- Программа по-прежнему пытается преобразовать любой набор данных, но сообщает о синтетических полях и возможной потере смысла.
- Исправлена пузырьковая диаграмма: данные X/Y больше не подменяются внутренними координатами раскладки.
- Новые узлы и серии получают новый цвет автоматически, если пользователь не выбрал цвет вручную.
- Автоматическая палитра действует в UI, импорте, обычном API и MCP-sidecar.
- Sankey дополнительно адаптирован к насыщенным демо: высота узлов и расстояния гарантированно помещаются в доступную область.

## Темы демо

| Диаграмма | Показательный сценарий |
|---|---|
| Network | Учёба, работа, здоровье, финансы, семья и покупки |
| Force | Команда и сервисы запуска интернет-магазина |
| Tree | Учебная программа по информационной безопасности |
| Mind Map | Личный план на год |
| Fishbone | Причины ухудшения здоровья и работоспособности |
| Flowchart | Оформление заказа в интернет-магазине |
| Decision | Выбор пути после университета |
| Sankey | Потоки покупателей интернет-магазина |
| Bubble | Ассортимент: маржа, спрос и выручка |
| Radar | Баланс жизни: текущее, желаемое и перегруженное состояние |

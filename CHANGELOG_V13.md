# Graph Studio v13

## Основные изменения

### Производительность

- фон и сетка кэшируются во вспомогательном Canvas;
- для крупных force/hex-графов используется пространственная сетка вместо полного перебора всех пар узлов;
- добавлена целевая частота кадров `performance.targetFps`;
- при падении FPS автоматически уменьшается количество фоновых частиц;
- hit-test мыши ограничен параметром `performance.hoverThrottleMs`;
- фон пересоздаётся только после изменения размера, темы или параметров сетки.

### Плавные переходы

- перед сменой типа диаграммы или раскладки сохраняется текущий кадр;
- новый режим проявляется через crossfade со `smoothstep`-сглаживанием;
- переход работает между Network, Tree, Fishbone, Flowchart, Sankey, Bubble, Radar и другими режимами;
- для режима Info дополнительно используется DOM-анимация поверхности.

### Редактор связи

- всплывающий редактор больше не открывается при наведении;
- редактирование открывается только после клика по связи;
- hover сохраняется только для подсветки и hit-test.

### Потоковые частицы

- хвост строится по всему маршруту связи, а не прямой хордой;
- хвост повторяет кривые, ломаные и ортогональные участки;
- увеличена стандартная длина хвоста;
- добавлено сегментированное затухание от головы к хвосту.

### Граф решений

- размер блоков рассчитывается с учётом формы;
- ромбы и круги получили уменьшенную внутреннюю область текста;
- длинные слова разбиваются на части;
- высота блока растёт до полного размещения подписи;
- минимальный размер шрифта уменьшен только как последняя мера.

### Информационные блоки

Добавлен машинно-понятный каталог из 17 типов:

`summary`, `ranking`, `kpi`, `table`, `timeline`, `alerts`, `checklist`, `comparison`, `progress`, `matrix`, `steps`, `pros_cons`, `glossary`, `recommendations`, `sources`, `quote`, `cards`.

Каталог доступен:

```js
graph.run('get_info_templates')
```

и через MCP-ресурс:

```text
graphstudio://contracts/info-templates
```

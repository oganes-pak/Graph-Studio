# Graph Studio v8

## Изменения

1. **Уникальная связь пары.** `A ↔ B` может существовать только один раз.
   Обратная запись `B → A` считается дубликатом.
2. **Live preview.** Настройки движения и оформления видны сразу. Сохранение в
   проект и LocalStorage происходит только после кнопки «Применить».
3. **Модульный просмотр.** Вместо ссылки отдельной темы используется
   `<graph-studio-widget mode="viewer">` или `embed.html`.
4. **Поток по ветвям.** Движущиеся элементы стали вытянутыми штрихами с хвостом.
5. **Сужение связей.** Сильная связь сохраняет толщину в середине, но плавно
   сужается около маленького узла.
6. **Редактор связи.** Изменения полей показываются в реальном времени,
   «Применить» фиксирует, «Отменить» возвращает исходное состояние.

# Смысловая совместимость диаграмм

Graph Studio может технически преобразовать любой поддерживаемый набор данных в любой тип диаграммы. Это не означает, что каждое такое представление будет одинаково естественным.

Примеры:

- **Tree** ожидает один корень и иерархию. Циклическая сеть будет сведена к уровням.
- **Sankey** ожидает направленные числовые потоки. При отсутствии `value` используется значение `1`.
- **Bubble** ожидает X, Y и размер. Если их нет, координаты выводятся из уровня и связности.
- **Radar** ожидает общие числовые метрики. Для обычного графа создаются структурные показатели.
- **Fishbone** ожидает проблему, категории и причины. Для произвольного графа категории определяются по первому уровню.

В интерфейсе показывается оценка от 0 до 100%:

- **80–100%**: данные естественно подходят диаграмме;
- **55–79%**: представление условное, часть полей вычисляется;
- **0–54%**: сильная интерпретация, пользователь должен проверить смысл результата.

Преобразование не блокируется. Пользователь получает предупреждения и может вручную уточнить значения после переключения.
